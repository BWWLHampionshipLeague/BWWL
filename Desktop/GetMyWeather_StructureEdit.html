This document will provide reference for using the GetMyWeather API and is intended for web developers. GetMyWeather allows you to embed a weather forecast into your page. GetMyWeather results are unique in that they are considered a "micro-forecast" and can be customized for a very specific location.

Overview

After initializing GetMyWeather, use the getWeather method for each forecast. The method getWeather returns an HTML snippet, which can be processed with Javascript and/or styled with CSS.

GetMyWeather is supported by all modern browswers that support Javascript, which must be enabled to use GetMyWeather. The return package is static, and you must invoke getWeather again with new parameter values for an update.

Prerequisites

Include this empty element in your HTML page. Your browser will insert the getWeather return package here.

<div id="forecast"></div>

Additionally, GetMyWeather will need initialized before the method can be called. The API needs initialized just once for a web session. Use the following:

<script type="text/javascript">
var weatherForecast;
function init() {
weatherForecast =
new getWeather(document.getElementById('forecast')}
</script>

Request Format and Results

To call getWeather use:

<script async defer
src="https://www.getmyweather.com/getWeather?
key=<YOUR_KEY>&
callback=init&
location=<LATITUDE>:<LONGITUDE>&
specificity=<SPECIFICITY>&
time=<TIME>">
</script>

Upon success, the getWeather function returns the following HTML structure (values are samples).

<div class="forecast">
<div class="temperature">78</div>
<div class="windspeed">15</div>
<div class="chanceRain">30</div>
<div class="trust">80</div>
</div>
